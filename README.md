# Alura Primeiro Projeto
Primeiro programa feito em C#, que uma versão menos complexa do sistema de um site de música chamado ScreenSound.

### Funcionalidades
- Registrar uma banda;
- Exibir as bandas registradas;
- Avaliar uma banda;
- Exibir a média da banda.
